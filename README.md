# Gestion_pedidos_inventario
Sistema que permite tomar pedidos y hacer control de inventarios principalmente para restaurantes 
Sistema de Gestión de Pedidos e Inventario

Este sistema permite administrar productos, gestionar inventario
y registrar pedidos mediante una aplicación web.

Funcionalidades principales:

- Gestión de productos
- Gestión de inventario
- Registro de pedidos
- Actualización automática del stock

Tecnologías utilizadas:

- Backend
- Base de datos SQL
- API REST
